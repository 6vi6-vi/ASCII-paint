# ASCII Paint

## Краткое описание
ASCII Paint - это редактор псевдографики для создания ASCII-арта в консоли. Программа позволяет рисовать линии, прямоугольники, выполнять заливку областей и сохранять результаты в файлы.

## Возможности
- Рисование линий и прямоугольников
- Заливка областей (flood fill)
- История действий (undo)
- Сохранение и загрузка в файлы .ascii
- Выбор символов для рисования
- Удобный интерфейс с цветовым выделением курсора

## Инструкция по сборке

### Требования
- CMake 3.10 или выше
- C++17 совместимый компилятор (MSVC, MinGW, GCC)
- Windows (используются WinAPI функции)

1. Клонируйте или распакуйте архив с проектом
